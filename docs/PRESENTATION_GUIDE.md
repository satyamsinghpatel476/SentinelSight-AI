# SentinelSight AI Presentation Guide

This guide will be expanded as each milestone lands. The current repository state is Milestone 1: project foundation.

## Three-Minute Demo Sequence

1. Open the React dashboard.
2. Confirm backend health and database readiness.
3. Open the controlled demo target in normal mode.
4. Explain that authentication, scanning, baselines, incidents and audit verification are upcoming milestones.

## Likely Judge Questions

### Is scanning implemented yet?

Not in Milestone 1. The foundation is intentionally small so security-sensitive scanning code can be added and tested milestone by milestone.

### Does the demo target serve malicious content?

No. The defaced mode is a controlled, test-only page with simulated suspicious text and an inert script reference.
